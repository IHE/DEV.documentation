## How This Site Works

This site is built automatically from Markdown files in the [DEV.documentation](https://github.com/IHE/DEV.documentation) repository. To update it:

1. Edit any `.md` file in the `docs/` directory (you can do this directly on GitHub)
2. Commit your changes to `main`
3. The site rebuilds automatically

No special tools needed. If you can edit a text file, you can update this site.
